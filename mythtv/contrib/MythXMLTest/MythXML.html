getProgramGuide - Retrieves the Program Guide for a given time frame.
StartTime:  
EndTime:  
StartChanId:  
NumOfChannels:  
Details: True False
* Note: maximum of 1000 programs returned per call
Params: StartTime   Starting Time (CCYY-MM-DDTHH:MM) <required>
  EndTime   EndTime must be after StartTime <required>
  StartChanId   Return guide from the provided Channel Id, if omitted, return from beginning.
  NumOfChannels   Number of sequencal channels to return for requested time period. If omitted, 1 channel is returned. -1 returns all channels.
  Details   Should all attributes be returned
