6. MySQL.

6.1 Mandrake 9.0

Make sure that mysql is running and started at boot. Click on Mandrake Control Center->System->Services, find mysql and click the "On Boot" button and the "Start" button if the mysql status shows that it isn't running yet.

6.2 Red Hat Linux 8.0

Make sure that mysql is running and started at boot. Click on GNOME menu>Server Settings>Services and enter the root password when asked. Check "mysqld" and then click Start. Click Save, then close the window.

6.3 Gentoo.

Please submit content.

6.4 Setting up the initial database.

Setup the initial mysql databases: 

% cd database

Mandrake 9.0

% mysql -u root < mc.sql

Debian 3.0

% mysql < mc.sql

Red Hat Linux 8.0

% mysql < mc.sql

if this returns an error message try switching to root first.

% su
# mysql < mc.sql
# exit
%
