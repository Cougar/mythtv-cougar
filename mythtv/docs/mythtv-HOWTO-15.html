15. Troubleshooting.

15.1 Illegal Instruction.

This error can happen for the following reasons:

  • You have an AMD K6-2 or VIA C3 processor. If you have either of these, make sure that you you followed the instructions in this section.
  • You have an Intel Celeron or Pentium Pro. If you have either of these processors, make sure that your deinterlace variable is set correctly.

15.2 mythfilldatabase failing.

If mythtvfilldatabase appears to be failing, edit the /usr/lib/perl5/site_perl/5.8.0/XMLTV/ZapListings.pm file and perform a global search and replace, searching for "tvlistings2" and replacing it with "tvlistings".

15.3 mythtv seems to disappear; can't see video

If you're using Gnome as your window manager, try switching to KDE. For Red Hat Linux 8.0 users, edit /etc/sysconfig/desktop and change "GNOME" to "KDE".
